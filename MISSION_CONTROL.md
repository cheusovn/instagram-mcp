## 🎨 ВИЗУАЛЬНЫЕ ПРАВИЛА

- Слайд 1: текст 80–100px, 1–2 строки максимум
- Цвета: NEON `#00F0FF` + MAGENTA `#FF2BD6` + тёмный фон `#080018`
- Анимации: каждый элемент влетает с задержкой 0.2s
- Нет стоковых фото, нет безликих иконок
- Нет слов: "эксперт", "уникальный", "качество"
- Без признаков ИИ-генерации в текстах

### Формулы Hook (слайд 1):
1. Провокация: "Ты делаешь X — и сам себе мешаешь"
2. Цифра: "3 фразы убили мои продажи. Вот они."
3. Тайна: "Один клиент платит 80к/мес за 2 часа работы"
4. Против течения: "Все советуют X. Я делаю наоборот."
5. Личная история: "Я облажался. Вот что вышло."
