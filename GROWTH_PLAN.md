## 🧠 СТРАТЕГИЯ

### Почему сейчас карусели не работают
Алгоритм Instagram 2026 года двигает контент по двум метрикам:
1. **Watch time** — досматривают ли до конца
2. **Shares** — пересылают ли в DM и Stories

Твои текущие карусели: 1-2 лайка → алгоритм считает их неинтересными и не показывает новым.
Твой вирусный пост (566 лайков) — Reel с личным видео.

### Формула роста для твоей ниши
```
ВИДЕОКАРУСЕЛЬ 4:5 (Remotion)
= Анимированные слайды как Reel
+ Музыка (trending audio)
+ Крючок в первые 2 секунды (текст на экране)
+ CTA → шапка профиля
```

### Почему 4:5, не 9:16
- 4:5 занимает максимум экрана в ленте → больше внимания
- Алгоритм одинаково продвигает 4:5 и 9:16 Reels
- Карусели в 4:5 выглядят как слайд-шоу Reel → Watch time выше

```bash
# Автоматически создаёт carousel.mp4 в 1080×1350
cd remotion-carousel && node render.mjs
```

### Шаг 3: Публикация
```bash
# Публикует через Instagram Graph API как Reel
node index.js publish_reel --file out/carousel.mp4 --caption "..."
```

### Что генерировать каждый день (30 минут работы)
1. `node multi-agent-factory.js "[тема утреннего поста]" --render` → публикуй в 10:00
