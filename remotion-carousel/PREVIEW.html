Style 02 + 04 → Liquid Glass Bento

Nano Banana генерирует premium bento-сетки с liquid glass (85-90% прозрачность, subtle motion) — идеально как фон-карточки для Candy Pop и Glassmorphism. Встраиваются через поле image в slides.json.

Style 01 + 06 → Cinematic Cover

Промпт "cinematic color grading, ultra-sharp, photorealistic, volumetric light" генерирует hero-кадры для обложки. Ken Burns + parallax поверх = вау-эффект.

Style 07 → Quote Cards

"Wide quote card, brown background, light-gold serif, large quotation mark, gradient transition" — точно в эстетику Swiss Minimal. Используй на слайде-цитате.

Style 03 + 05 → Sci-Fi / Cyberpunk

Nano Banana имеет категорию Cyberpunk/Sci-Fi с neon-фонами. Генерируй через него AI-арт и вставляй как VideoBackground / KenBurnsImage за текстом.

Все стили → Product Infographic

Bento-промпт создаёт 8-модульную сетку с hero-color деривацией. Встраивай как слайд с данными/сравнениями нейросетей вместо текстового body.

Вердикт по сочности

Nano Banana Pro наиболее сочен в паре с Style 02 (Candy), Style 04 (Glass), Style 05 (Blobs) — там liquid glass и motion hint прямо в промптах. Style 01 и 06 — кинематограф.
